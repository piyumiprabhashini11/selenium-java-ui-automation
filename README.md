# Selenium Java UI Automation

This repository contains practical Selenium Web UI automation examples using Java and Maven.
It covers essential automation topics such as:
- Dropdown handling
- Waits (implicit, explicit, fluent)
- Alerts
- Iframes
- Web element interactions
- Basic browser actions

## Tech Stack

- Selenium 4
- Java 11+
- Maven
- TestNG
- Eclipse IDE or IntelliJ
- Git & GitHub

## Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/piyumiprabhashini11/selenium-java-ui-automation.git
```
### 2. Prerequisites

Make sure the following tools are installed on your system:

Java 11+

Maven

Git

Eclipse IDE or IntelliJ IDEA

### 3. Verify installations:

```bash
java -version

mvn -version
```
